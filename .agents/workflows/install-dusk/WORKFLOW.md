# Install Dusk

## Preparation

1. Request git clone access from human.
2. Find `https://github.com/head-js/Dusk` and clone it into the `.context` directory.

```bash
mkdir -p .context
git clone git@github.com:head-js/Dusk.git .context/dusk
```
